# 小众软件

## 1、NohBoard
NohBoard 是一款键盘可视化程序，它能改在屏幕上显示整个虚拟键盘，并实时、动态、高亮显示按下的键。有多种虚拟键盘布局（游戏键盘等）、多种皮肤。
对于秀快捷键、直播打游戏什么的很合适。

https://github.com/ThoNohT/NohBoard
