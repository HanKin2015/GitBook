# AutoHotkey
参考：https://zhuanlan.zhihu.com/p/103357456


自动化脚本语言，简单说，它可以帮助完成大量重复性的工作。

```
/*温馨提示*/
;Windows系统默认的Win快捷键:
;Win + E：打开资源管理器;
;Win + D：显示桌面;
;Win + F：打开查找对话框;
;Win + R：打开运行对话框;
;Win + L：锁定电脑;
;Win + PauseBreak：打开系统属性对话框;
;Win + Q: 本地文件 / 网页等搜索;
;Win + U: 打开控制面板－轻松使用设置中心;
;Shift+Ctrl+Esc: 任务管理器;
```
