# rz上传sz下载

## 1、安装

yum -y install lrzsz


## 2、mobaxterm使用sz下载

sz  filename
ctrl + 鼠标右键
右键菜单栏中选择：Receive file using Z-modem

## 3、mobaxterm使用rz上传

rz
ctrl + 鼠标右键
右键菜单栏中选择：Send file using Z-modem

## 4、一直想知道的单词缩写
Receive file using Z-modem(rz)
Send file using Z-modem(sz)

这样理解就能说通：
在服务器中执行命令，sz相当于服务端要发送文件，即下载。rz服务端要接收文件，即上传。
