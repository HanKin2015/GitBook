# 1、标题栏显示完成路径

 Options -> Preferences -> Display -> Trim long path names with ellipses.(把复选框的勾选去掉。) 

# 2、显示空格和tab

 View --> Visible Tabs and spaces。把“ Visible Tabs and spaces”勾选上就行了。 

# 3、设置字体

Options-》File Type Options

还原界面布局：View-》Reset Layout
