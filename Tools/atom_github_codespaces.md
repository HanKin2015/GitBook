## 1、atom （一款开源的代码编辑器） 
Atom 是github专门为程序员推出的一个跨平台文本编辑器。具有简洁和直观的图形用户界面，并有很多有趣的特点：支持CSS，HTML，JavaScript等网页编程语言。它支持宏，自动完成分屏功能，集成了文件管理器。
Atom是由GitHub开发的自由及开放源代码的文字与代码编辑器。

官网：https://atom-china.org/
