# 强大的git bash软件

## 功能一
git是程序员必需品，因此git bash也是必需的。

## 功能二
git bash是一个不错的Linux终端，可以在Windows下畅所欲言的使用各种bash命令。



这样就无需再去安装其他Linux终端，如。

## 升级更新
输入：git update-git-for-windows
国内镜像站下载：https://github.com/waylau/git-for-win
